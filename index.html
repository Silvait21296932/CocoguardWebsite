Project Scope

Literature Survey

Pest Detection (Coconut Mite)

Recent studies have applied CNNs like MobileNetV2 for pest detection, yet they often rely on post-harvest, high-resolution images under controlled conditions. CocoGuard addresses this gap by combining Real-ESRGAN for image enhancement with MobileNetV2 to detect Aceria guerreronis (coconut mite) infestations directly from mobile phone images in the field.

Leaf Disease Detection (Grey Leaf Spot)

While CNNs have shown success in plant disease detection, limited research exists for coconut-specific foliar diseases like Grey Leaf Spot. This component integrates MobileNetV2 with explainable AI (Grad-CAM++ and LIME) to enhance interpretability, improving trust and usability of mobile-based disease detection systems in real-world agricultural environments.

Maturity & Harvest Prediction

Existing models classify coconut maturity using CNNs but lack predictive capabilities for optimal harvest scheduling. This component merges image-based maturity detection via MobileNetV2 with Random Forest regression on historical yield data, offering farmers a comprehensive tool for real-time maturity assessment and precise harvest time prediction.

Soil Fertility Optimization

Conventional coconut farming often misuses fertilizers due to a lack of real-time nutrient data. This module employs IoT sensors to collect NPK, pH, and temperature values, feeding a Random Forest model to predict precise fertilizer needs. It bridges the gap between blanket guidelines and field-specific, data-driven recommendations.

Research Gap

Multi-Domain Identification & Classification

There is no unified mobile-based solution in Sri Lanka for detecting coconut pests, leaf diseases, nutrient deficiencies, and fruit maturity stages. Most existing approaches target a single issue in isolation, and visually similar symptoms make field-level AI classification especially challenging without domain-specific tuning.

Actionable Prediction & Timely Intervention

Current research often stops at detection, lacking predictive insights such as optimal harvest time or required fertilizer quantities. There is limited integration of environmental, historical, or image-based data to support early intervention strategies that improve long-term crop yield and reduce preventable losses.

Real-Time Data Use & Farmer Communication

Many systems fail to provide real-time, location-specific guidance or alert mechanisms for farmers. There is a lack of accessible platforms that integrate image uploads, sensor readings, and explainable AI into a user-friendly mobile application, limiting effective communication between farmers and agricultural experts.

Research Problem & Solution

Proposed Problem

How can we accurately detect pests, diseases, nutrient imbalances, and maturity stages in coconut farming to improve productivity and decision-making for Sri Lankan farmers?

Addressing the critical issues in coconut cultivationâ€”including pest infestation, foliar diseases, soil mismanagement, and inconsistent harvest timingâ€”requires a unified, intelligent system. Traditional methods are labor-intensive, error-prone, and lack scalability. Early detection and personalized insights are essential to reduce crop loss, improve yield, and promote sustainable farming practices.

Proposed Solution

CocoGuard presents an end-to-end AI and IoT-based mobile platform that empowers farmers with real-time monitoring and intelligent insights for coconut farming. It integrates four smart modules: 1. Pest Detection: Enhances in-field nut images using Real-ESRGAN and classifies coconut mite infestations with MobileNetV2. 2. Disease Identification: Detects Grey Leaf Spot with high accuracy using CNNs and boosts interpretability using Grad-CAM++ and LIME. 3. Soil Fertility Management: Employs IoT sensors and Random Forest models to provide precise NPK fertilizer recommendations based on live field data. 4. Harvest Prediction: Combines image-based maturity classification with yield history using Random Forest regression to suggest optimal harvest timing. This unified system runs via a React Native app and Flask backend, ensuring real-time image processing, field data analysis, and smart alerts. By combining deep learning, explainable AI, and IoT, CocoGuard transforms traditional coconut farming into a data-driven, efficient, and sustainable agricultural model.

Research Objectives

01

Pest Detection (Coconut Mite â€“ Ravindu Balasooriya)

The main objective is to develop an intelligent mobile-based system for early detection of coconut mite infestations directly on trees using field images. The system enhances low-quality images captured by farmers using Real-ESRGAN and classifies them with a lightweight MobileNetV2 model. It aims to minimize crop damage by providing timely identification of Aceria guerreronis infestations. Additional goals include improving model robustness through data augmentation, deploying predictions via a user-friendly app, and ensuring the model functions effectively under varying lighting and camera conditions to support widespread, real-time use in low-resource agricultural settings.

02

Leaf Disease Detection (Grey Leaf Spot â€“ Gunarathne M.M.S.U.)

The objective is to automate the detection of Grey Leaf Spot, a visually subtle coconut leaf disease, using a deep learning-based mobile solution. A MobileNetV2 model classifies leaf images into healthy, infected, or non-coconut categories. To address trust and transparency, the system integrates Grad-CAM++ and LIME for visual explanation of predictions. The goal is to provide early, accurate disease identification while enhancing farmer trust in AI-based systems. Additional objectives include dataset creation using both field and synthetic images, optimizing the model for mobile deployment, and building an intuitive frontend that supports efficient image uploading and result interpretation.

03

Maturity & Harvest Prediction (Silva T.C.D.)

The objective is to create a hybrid system that not only classifies the maturity level of coconuts through image-based analysis but also predicts the ideal harvest date using historical data. MobileNetV2 is used for real-time image classification, while a Random Forest regression model is trained on user-inputted harvest records such as yield volume, date, and location. The system aims to help farmers make data-driven decisions for maximizing harvest quality and reducing premature or delayed harvesting. Additional objectives include field-level data collection, model training, app integration, and presenting customized recommendations via a mobile interface for usability in rural areas.

04

Soil Fertility Optimization (Adithya E.L.A.Y.)

The primary objective is to develop an IoT-enabled solution for real-time soil monitoring and nutrient recommendation in coconut farming. A custom device collects NPK, pH, temperature, and humidity data using sensors, which is processed by a Random Forest model to provide fertilizer guidance specific to each field. The system addresses the lack of localized nutrient insights and aims to reduce overuse or underuse of fertilizers. Supporting objectives include comparing multiple ML models (XGBoost, Gradient Boosting), optimizing sensor accuracy, integrating a React Native app for user interaction, and promoting precision agriculture through sustainable fertilizer application practices.
